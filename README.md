# SIAKNG-Lite Frontend 

Repositori ini berisi aplikasi frontend untuk SIAKNG-Lite, sistem informasi akademik berbasis web untuk mendukung aktivitas perkuliahan.
Aplikasi ini menyediakan alur autentikasi, dashboard berdasarkan peran (Dosen/Mahasiswa), manajemen mata kuliah dan kelas, pengisian IRS, serta ringkasan studi dan nilai.

## Tech Stack

- Framework: React.js (TypeScript)

- Styling: Tailwind CSS

## Deployment

- Aplikasi dapat diakses di: [https://siakng-lite.vercel.app/](https://siakng-lite.vercel.app/)

## Menjalankan Secara Lokal

1. Clone repository:
	git clone https://github.com/<username>/siakng-lite-frontend.git

2. Masuk ke folder project:
	cd siakng-lite-frontend

3. Install dependency:
	npm install

4. Jalankan development server:
	npm run dev

5. Buka di browser:
	http://localhost:5173
